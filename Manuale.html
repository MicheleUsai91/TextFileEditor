Benvenuto nel Dynamic TXT File Editor. Questo strumento è stato progettato per elaborare, correggere e unire file TXT e CSV in modo massivo o singolo.

1. Elaborazione Massiva

Il cuore del sistema. Questa funzione automatizza l'intero processo:

  • Legge tutti i file TXT presenti nella cartella ORIGINALI.
  • Arricchisce automaticamente i dati trovando le corrispondenze nei file all'interno della cartella ESITI.
  • Esporta il risultato finale, pronto e formattato, nella cartella DEFINITIVI.

2. Elaborazione Singola

Se devi lavorare su file specifici fuori dal flusso massivo:

  • Converti Excel in CSV: Lancia uno script PowerShell locale per trasformare i file Excel in formato CSV leggibile dal sistema.
  • Carica TXT: Carica un singolo file TXT per visualizzarlo nella tabella.
  • Completa la Tabella: Una volta caricato un TXT, puoi caricare un CSV per iniettare gli esiti nelle relative colonne.

3. Gestione Tabella e Dati

Una volta che i dati sono caricati nella tabella in basso, hai diverse opzioni di modifica:

  • Modifica Diretta (Inline): Clicca direttamente sulle celle delle colonne EER_ESITO e EER_NOTE_RIENTRO per digitare nuovi valori. Il sistema applicherà la spaziatura (padding) corretta in automatico al salvataggio.
  • Eliminazione Massiva: Usa il box "Elimina da lista ID" per incollare una lista di ID e cancellare in un colpo solo tutte le righe corrispondenti.
  • Ricerca e Filtri: Usa la barra di ricerca per trovare rapidamente stringhe nei campi principali (Pratica, Codice Fiscale, Esito).

4. Strumenti ed Esportazione

Dal pannello strumenti puoi generare i file finali:

  • Esporta in CSV: Salva l'intera tabella visualizzata in un file all'interno della cartella CSV.
  • Export TXT: Genera i file TXT strutturati e li salva nella cartella MODIFICATI (pronti per essere uniti).

5. Unione dei File (Merge)

L'ultimo passaggio per assemblare i lotti. Il sistema legge tutti i file presenti in MODIFICATI. Spunta i file che desideri combinare e clicca "Unisci file selezionati". Il file combinato verrà creato nella cartella UNITI.
